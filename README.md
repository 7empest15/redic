# Redic

Redic is a minimalistic implementation of Redis written in C. This project is designed purely for educational purposes and should not be used in real-world applications.

## Features
- Basic caching functionality
- Simple hash table implementation
- Network handling for basic communication

## Project Structure
- `src/`: Contains the source code files
- `includes/`: Header files for the project
- `obj/`: Compiled object files

## Getting Started

### Prerequisites
- A C compiler (e.g., `gcc`)
- Make

### Build Instructions
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd redic
   ```
2. Build the project using the Makefile:
   ```bash
   make
   ```

### Running the Project
After building, you can run the project with:
```bash
./redic
```
## Disclaimer
This project is a learning exercise and is not intended for production use. It lacks the robustness, security, and features of a full-fledged Redis implementation.
